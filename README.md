# Fruit vs Vegetable Recipe Classifier

This is a simple PyTorch text classification app.

It predicts whether a recipe name belongs to the fruit category or vegetable category.

## Files

- app.py: Streamlit GUI and PyTorch model
- recipes_fruit_veg.csv: Dataset
- requirements.txt: Python dependencies
- Dockerfile: Docker deployment file

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
